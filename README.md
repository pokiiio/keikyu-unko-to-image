# keikyu-unko-to-image
京急の運行情報を取得し画像化します。

## 使い方

特に深い意味はないですが、Python3の環境で動作させることを想定しています。また、requestsモジュールを使っているので、予めインストールをお願いします。


```bash
$ pip3 install requests
```

あとは実行するだけです。


```bash
$ python3 keikyu-unko-to-image.py
```

おしまい。
